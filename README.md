# Web-Based Stock Portfolio Tracker

A dynamic Web Application built using Python and Flask that allows users to manage and track their stock portfolios in real-time through a clean web interface.

## Features
* **Web UI:** No more boring terminal! Track stocks via a beautiful web page.
* **Dropdown Selection:** Easy selection of predefined stocks (AAPL, TSLA, MSFT, etc.).
* **Dynamic Calculations:** Instantly displays the updated total value of individual holdings and the entire portfolio.
* **Reset Functionality:** Clear the entire portfolio with a single click.

## Tech Stack
* **Backend:** Python 3.x, Flask Framework
* **Frontend:** HTML5, CSS3
